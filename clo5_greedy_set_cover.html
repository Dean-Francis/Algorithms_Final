Educational Overview Set cover problem, greedy approximation, and complexity
›

The Set Cover problem asks: given a universe U of n elements and a family F of subsets, find the fewest subsets from F whose union equals U. It is NP-hard, so we use a greedy approximation that runs in polynomial time.

Greedy Rule
At each round, pick the set S ∈ F that covers the most uncovered elements. For weighted set cover, pick the set with the best |S ∩ R| / w(S) ratio (coverage per unit cost).
Approximation Ratio
Let k = max |S| (largest set size). Greedy produces a cover of size at most H(k) · OPT, where H(k) = 1 + 1/2 + … + 1/k ≈ ln k.

This is essentially tight: unless P = NP, no polynomial algorithm achieves ratio better than (1 − ε) ln n (Feige, 1998).
Complexity
Decision version: NP-Complete (Karp, 1972).
Greedy algorithm: O(n · |F| · max|S|) — polynomial.
Vertex Cover is a special case: each edge = element, each vertex = set of its incident edges.
